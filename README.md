## Title: 

Photo Mode - a simple THREE.JS app

## Description: 

You can see this as a (mega) simplified version of Forza Horizon 4's Photo Mode.

## Controls:


- Left Click to PAN
- Right Click to ROTATE
- Mouse Wheel to ZOOM
+ scene settings in the GUI in the upper-right corner of the screen
